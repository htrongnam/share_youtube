# share_youtube

## Dependencies
- Ruby 2.5.1
- Rails 5.2.0
- Json Web Token (JWT)
- PostgreSQL
- Bcrypt
- Cloudinary
- Carrierwave

## Installation
- Clone this project

        $ git clone https://github.com/htrongnam/share_youtube.git
        
- Change directory to this project

        $ cd share_youtube

- Install dependencies

        $ bundle install

- Create database

        $ rake db:create
        
- Run migration

        $ rake db:migrate
        
- Run this project
        
        $ rails server
